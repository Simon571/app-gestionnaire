Installer l'application — guide simple

Ce guide est imprimable et conçu pour les personnes non techniques.

Étapes (3 clics)

  1. Téléchargez le fichier .msi depuis la page officielle.
  2. Double‑cliquez le fichier téléchargé et suivez l'assistant d'installation.
  3. Ouvrez le menu Démarrer → tapez « Gestionnaire d'Assemblée » → lancez l'application.

Si Windows affiche un avertissement

Cliquer sur Plus d'infos → puis Exécuter quand même si vous avez téléchargé depuis la page officielle.

Vérification (optionnel)

Ouvrez PowerShell et exécutez :

Get-FileHash $HOME\Downloads\nom-fichier.msi -Algorithm SHA256

Comparez la valeur affichée avec la SHA‑256 fournie sur la page.

Besoin d'aide ?

Envoyez une capture d'écran et le nom du fichier téléchargé au support (bouton "Signaler un problème" sur la page).
