# **App Name**: Local Assembly Manager

## Core Features:

- Dashboard: Dashboard for quick overview of key metrics (e.g., active publishers, meeting attendance).
- Assembly Registration: Form for managing assembly information: name, ID, meeting times, Zoom details, contact information
- Publisher Directory: Maintain directory of publishers with details like name, contact, territory assignment.
- AI Auto-Logger: Tool to create a form to automatically log congregational activities. LLM reasons which parts of the previous activities have changed, to make suggestions to the admin user, helping the admin to avoid tedious, repetitive data entry.
- Territory Assignment: Territory management: assign territories and track completion status.
- Report Generation: Report generator: create PDF/PNG reports of publisher activity and meeting attendance.
- Data Management: Import/Export data in CSV/Excel formats and database backups

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and stability. The French flag may have subtly influenced this choice, while avoiding any literal resemblance.
- Background color: Light Gray (#E8EAF6) to maintain a professional, clean aesthetic and a quiet feeling. 
- Accent color: Purple (#7E57C2) as an analogous color to draw attention to interactive elements and important actions. This may reflect the religion suggested by the prompt.
- Body and headline font: 'PT Sans' (sans-serif) blends modern look and a little warmth.
- Simple, professional icons from a library like Font Awesome to maintain clarity and consistency.
- Clean, card-based layout with Bootstrap 5 grid system for responsiveness and clear information hierarchy.
- Subtle transitions and animations on interactive elements for user feedback (e.g., button hover effects).
